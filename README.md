# Educational Institution Website

This repository contains the complete source code for an educational institution's website, featuring various sections to provide comprehensive information about the university and its offerings. The site is designed to serve students, faculty, alumni, and prospective students with easy access to important information.

## Features

- Responsive design for mobile and desktop viewing
- Multiple dedicated pages for different audiences
- SEO-friendly structure
- Clean, modern interface appropriate for educational institutions

## Pages

The website includes the following pages:

- **Home** (`index.html`) - Main landing page with overview of the institution
- **About** (`about.html`) - Detailed information about the institution's history, mission, and values
- **For Universities** (`for-universities.html`) - Information targeted at university partnerships and collaborations
- **Universities** (`universities.html`) - List of partner universities and exchange programs
- **Abiturients** (`abiturients.html`) - Information for prospective students, admission requirements, and application process
- **Alumni** (`alumni.html`) - Alumni resources, networking opportunities, and reunion information
- **Career** (`career.html`) - Career services, job placement, and professional development
- **Housing** (`housing.html`) - Student accommodation options and residential life
- **Science** (`science.html`) - Research programs, laboratories, and academic achievements
- **Benefits** (`benefits.html`) - Benefits offered to students, faculty, and staff
- **Features** (`features.html`) - Key features of the institution's academic and extracurricular programs
- **Ecology** (`ecology.html`) - Environmental programs, sustainability initiatives, and green campus efforts
- **Partners** (`partners.html`) - Information about institutional partners and corporate relationships
- **Demo** (`demo.html`) - Interactive demonstration page
- **Universities 1** (`universities1.html`) - Additional universities information and academic partnerships

## Technology Stack

- HTML5 for structure
- CSS3 for styling
- JavaScript for interactivity
- Responsive design principles
- Optimized for GitHub Pages deployment

## Setup

To run this website locally:

1. Clone the repository
2. Open any HTML file in your browser or serve the directory using a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Or using Node.js with http-server
   npx http-server
   ```

## Domain Configuration

The repository includes a `CNAME` file, indicating this site is configured for custom domain hosting on GitHub Pages. The custom domain is specified in the CNAME file.
