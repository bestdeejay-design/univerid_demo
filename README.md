# Educational Institution Website

This repository contains the source code for an educational institution's website, featuring various sections to provide comprehensive information about the university and its offerings.

## Pages

The website includes the following pages:

- **Home** (`index.html`) - Main landing page
- **About** (`about.html`) - Information about the institution
- **For Universities** (`for-universities.html`) - Information targeted at university partnerships
- **Universities** (`universities.html`) - List of partner universities
- **Abiturients** (`abiturients.html`) - Information for prospective students
- **Alumni** (`alumni.html`) - Alumni information and resources
- **Career** (`career.html`) - Career services and opportunities
- **Housing** (`housing.html`) - Accommodation information
- **Science** (`science.html`) - Research and science programs
- **Benefits** (`benefits.html`) - Benefits offered by the institution
- **Features** (`features.html`) - Key features of the institution
- **Ecology** (`ecology.html`) - Environmental programs and initiatives
- **Partners** (`partners.html`) - Information about institutional partners
- **Demo** (`demo.html`) - Demonstration page
- **Universities 1** (`universities1.html`) - Additional universities information

## Setup

To run this website locally:

1. Clone the repository
2. Open any HTML file in your browser or serve the directory using a local server

## Domain Configuration

The repository includes a `CNAME` file, suggesting this site is configured for custom domain hosting on GitHub Pages.
