# Шаблон страницы UniveriD

## Описание

Файл `template.html` содержит общий шаблон для всех страниц сайта UniveriD. Шаблон включает:

- Общие стили (CSS)
- Верхнюю часть страницы (header) с навигацией
- Нижнюю часть страницы (footer)
- Функциональность переключения темы (светлая/темная)
- Мобильное меню
- Сохранение выбранной темы в localStorage

## Структура шаблона

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <!-- Метаданные, стили и заголовок -->
</head>
<body>
  <div class="container">
    <header>
      <!-- Логотип, навигация и переключатель темы -->
    </header>

    <!-- Основное содержимое страницы -->
    <main id="mainContent">
      <!-- Сюда добавляется уникальное содержимое каждой страницы -->
    </main>
  </div>

  <footer>
    <!-- Футер сайта -->
  </footer>

  <script>
    <!-- JavaScript для переключения темы и мобильного меню -->
  </script>
</body>
</html>
```

## Как создать новую страницу на основе шаблона

1. Скопируйте содержимое `template.html` в новый файл
2. Измените заголовок страницы в теге `<title>`
3. Добавьте уникальное содержимое внутрь тега `<main id="mainContent">`
4. При необходимости добавьте уникальные стили или скрипты

### Пример новой страницы:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
  <title id="pageTitle">UniveriD — Новости</title>
  <!-- Все стили из шаблона -->
  <style>
    /* Общие стили из шаблона */

    /* Уникальные стили для страницы новостей */
    .news-item {
      padding: 20px;
      border-bottom: 1px solid var(--border);
    }
  </style>
</head>
<body>
  <div class="container">
    <header>
      <!-- Заголовок и навигация из шаблона -->
    </header>

    <main id="mainContent">
      <h1>Новости UniveriD</h1>
      <div class="news-item">
        <h2>Новый функционал</h2>
        <p>Добавлена возможность бронирования аудиторий</p>
      </div>
      <!-- Другое уникальное содержимое страницы -->
    </main>
  </div>

  <footer>
    <!-- Футер из шаблона -->
  </footer>

  <script>
    <!-- Общие скрипты из шаблона -->
    
    <!-- Уникальные скрипты для страницы новостей -->
  </script>
</body>
</html>
```

## Особенности

- Переключение темы сохраняется между страницами через localStorage
- Все страницы имеют согласованный внешний вид
- Мобильное меню работает на всех страницах
- Для правильной работы темы у всех страниц должен быть одинаковый ID у элементов переключения темы
